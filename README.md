# ⛪ Church Management System

![License](https://img.shields.io/badge/License-MIT-green.svg)
![Status](https://img.shields.io/badge/Status-In%20Development-yellow)
![Tech](https://img.shields.io/badge/Stack-Flutter%20%7C%20Golang%20%7C%20PostgreSQL-blue)

A **web-based platform** to streamline church operations — manage members, track attendance, schedule events, send announcements, and handle finances.  
Built with **Next.js**, **Express.js**, **PostgreSQL**, and deployed as a **PWA** for easy cross-platform access.

---

## ✨ Features
- 📋 **Member Management** – Detailed profiles & ministry involvement  
- 📊 **Attendance Tracking** – Analytics for services & events  
- 📅 **Event Scheduling** – Create & share upcoming events  
- 💬 **Communication Tools** – Announcements & reminders  
- 💰 **Financial Management** – Donations & transparent reporting  
- 🔐 **Role-Based Access** – Permissions for admins, pastors & volunteers  

---

## 🛠 Tech Stack
| Layer       | Technology |
|-------------|------------|
| **Frontend** | Next.js, TypeScript, Material-UI |
| **Backend** | Express.js, Node.js, Prisma ORM |
| **Database** | PostgreSQL + PostGIS |
| **Deployment** | Progressive Web App (PWA) |
| **Security** | Helmet.js |
| **Logging** | Morgan |
